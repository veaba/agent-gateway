# 第 9 章：GUI 使用

## 启动 GUI

### Windows

```bash
# 双击运行
agw-gui.exe

# 或命令行
agw-gui
```

### macOS/Linux

```bash
./agw-gui
```

## 界面概览

### 系统托盘

启动后，GUI 会最小化到系统托盘：

- **托盘图标**：显示网关状态
- **右键菜单**：
  - 显示窗口
  - 启动网关
  - 停止网关
  - 退出

### 窗口结构

```
┌─────────────────────────────────────────────┐
│  agent-gateway                        [─][□][×]│
├─────────────────────────────────────────────┤
│ [仪表盘] [套餐] [Provider] [Fallback] [日志] │
├─────────────────────────────────────────────┤
│                                             │
│              内容区域                        │
│                                             │
└─────────────────────────────────────────────┘
```

## 仪表盘

显示网关运行状态：

- **状态**：运行中/已停止
- **当前套餐**：正在使用的套餐
- **配额使用**：今日/本月使用量
- **请求统计**：今日请求数、错误数、Fallback 次数

## 套餐管理

### 添加套餐

1. 点击「添加套餐」按钮
2. 选择 Provider
3. 输入 API Key
4. 选择模型
5. 点击「保存」

### 编辑套餐

1. 选中套餐
2. 点击「编辑」
3. 修改配置
4. 点击「保存」

### 删除套餐

1. 选中套餐
2. 点击「删除」
3. 确认删除

## Provider 浏览器

### 浏览 Provider

1. 点击「Provider」标签
2. 查看所有可用 Provider
3. 查看套餐模板
4. 查看模型能力

### 查看详情

1. 点击 Provider 名称
2. 查看套餐列表
3. 查看模型详情
4. 查看 Agent 支持

## Fallback 配置

### 启用 Fallback

1. 点击「Fallback」标签
2. 勾选「启用自动 Fallback」
3. 设置优先级
4. 设置最大尝试次数
5. 点击「保存」

## 日志查看

### 查看日志

1. 点击「日志」标签
2. 选择日志级别
3. 选择时间范围
4. 查看日志详情

### 筛选日志

- 按套餐筛选
- 按级别筛选
- 按时间筛选
- 搜索关键字

## 剪贴板监控

GUI 会自动监控剪贴板，当检测到 API Key 时：

1. 弹出提示
2. 选择套餐
3. 自动保存

支持格式：
- `sk-` (OpenAI)
- `sk-ant-` (Anthropic)
- `sk-proj-` (OpenAI 项目)
- `AIza` (Google)
- `gsk_` (Google AI Studio)
- `kilo_` (Kilo Code)

## 设置

### 常规设置

- 启动时运行
- 最小化到托盘
- 开机启动

### 通知设置

- 配额警告通知
- Fallback 通知
- 错误通知

### 代理设置

- HTTP 代理
- HTTPS 代理
- 跳过证书验证

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+1 | 仪表盘 |
| Ctrl+2 | 套餐 |
| Ctrl+3 | Provider |
| Ctrl+4 | Fallback |
| Ctrl+5 | 日志 |
| Ctrl+Q | 退出 |
| Ctrl+H | 显示/隐藏 |

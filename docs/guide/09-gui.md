# 第 9 章：GUI 使用

## 启动 GUI

### Windows

```bash
# 双击运行
agw-gui.exe

# 或命令行
agw-gui
```

### macOS/Linux

```bash
./agw-gui
```

## 服务器模式

GUI 默认以**嵌入式模式**启动，内置完整服务器同时提供 Proxy 和管理 API。

### 嵌入式模式

- GUI 启动时自动联动启动服务器
- 默认监听地址：`127.0.0.1:8080`
- 同时提供 Proxy (`/v1/*`) 和管理 API (`/api/v1/*`)
- 无需额外部署，开箱即用

### 外部模式

适用于连接远程或团队共享的 API 服务器：

1. 在设置中切换到「外部服务器」
2. 输入外部服务器地址（如 `http://192.168.1.100:8081`）
3. 测试连接确保可达
4. 保存配置

### 验证服务器运行

```bash
# 嵌入式模式
curl http://127.0.0.1:8080/health

# 管理 API 测试
curl http://127.0.0.1:8080/api/v1/plans
```

## 界面概览

### 系统托盘

启动后，GUI 会最小化到系统托盘：

- **托盘图标**：显示网关状态
- **右键菜单**：
  - 显示窗口
  - 启动网关
  - 停止网关
  - 退出

### 窗口结构

```
┌─────────────────────────────────────────────┐
│  agent-gateway                        [─][□][×]│
├─────────────────────────────────────────────┤

├─────────────────────────────────────────────┤
│                                             │
│              内容区域                        │
│                                             │
└─────────────────────────────────────────────┘
```

## 仪表盘

显示网关运行状态：

- **服务器模式**：嵌入式/外部
- **状态**：运行中/已停止
- **监听地址**：当前服务器地址
- **当前套餐**：正在使用的套餐
- **配额使用**：今日/本月使用量
- **请求统计**：今日请求数、错误数、Fallback 次数

## 套餐管理

### 添加套餐

1. 点击「添加套餐」按钮
2. 选择 Provider
3. 输入 API Key
4. 选择模型
5. 点击「保存」

### 编辑套餐

1. 选中套餐
2. 点击「编辑」
3. 修改配置
4. 点击「保存」

### 删除套餐

1. 选中套餐
2. 点击「删除」
3. 硋认删除

## Provider 浏览器

### 浏览 Provider

1. 点击「Provider」标签
2. 查看所有可用 Provider
3. 查看套餐模板
4. 查看模型能力

### 查看详情

1. 点击 Provider 名称
2. 查看套餐列表
3. 查看模型详情
4. 查看 Agent 支持

## Fallback 配置

### 启用 Fallback

1. 点击「Fallback」标签
2. 勾选「启用自动 Fallback」
3. 设置优先级
4. 设置最大尝试次数
5. 点击「保存」

## 日志查看

### 查看日志

1. 点击「日志」标签
2. 选择日志级别
3. 选择时间范围
4. 查看日志详情

### 篮选日志

- 按套餐筛选
- 按级别筛选
- 按时间筛选
- 搜索关键字

## 剪贴板监控

GUI 会自动监控剪贴板，当检测到 API Key 时：

1. 弹出提示
2. 选择套餐
3. 自动保存

支持格式：
- `sk-` (OpenAI)
- `sk-ant-` (Anthropic)
- `sk-proj-` (OpenAI 项目)
- `AIza` (Google)
- `gsk_` (Google AI Studio)
- `kilo_` (Kilo Code)

## 设置

### 服务器设置

- **服务器模式**：嵌入式/外部
- **嵌入式监听地址**：如 `127.0.0.1:8080`
- **外部服务器地址**：如 `http://192.168.1.100:8081`
- **自动启动**：GUI 启动时自动启动服务器

### 常规设置

- 启动时运行
- 最小化到托盘
- 开机启动

### 通知设置

- 配额警告通知
- Fallback 通知
- 错误通知

### 代理设置

- HTTP 代理
- HTTPS 代理
- 跳过证书验证

## 配置文件位置

所有配置统一存储：

```
~/.agent-gateway/
├── agw.yaml                    # 公共配置
├── agw-core/                   # Core 数据
│   ├── user_plans.yaml         # 套餐配置
│   ├── fallback.yaml           # Fallback 配置
│   └── gateway.db              # SQLite 数据库
├── agw-cli/                    # CLI 数据
│   └── config.yaml
└── agw-gui/                    # GUI 数据
    ├── config.yaml             # GUI 配置
    ├── tray.yaml               # 托盘配置
    └── server.yaml             # 服务器模式配置
```

环境变量覆盖：`AGENT_GATEWAY_HOME=/custom/path`
